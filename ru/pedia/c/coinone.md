# Coinone

## Обзор
Coinone — это известная южнокорейская платформа криптовалютной биржи, предлагающая различные услуги торговли цифровыми активами. Запущенная в 2014 году, Coinone стремится обеспечить безопасный и удобный опыт для торговли криптовалютами. Платформа выросла и стала одной из ведущих бирж в Южной Корее, предлагая различные функции, включая спотовую торговлю, маржинальную торговлю и надежный API для алгоритмической торговли. Coinone известна своими строгими протоколами безопасности и соответствием нормативным требованиям, что делает её авторитетным игроком в криптопространстве.

## Функции

### Спотовая торговля
Coinone позволяет пользователям торговать широким спектром криптовалют, включая, помимо прочего, Bitcoin (BTC), Ethereum (ETH), Ripple (XRP) и Litecoin (LTC). Платформа предоставляет удобный интерфейс с рыночными данными в режиме реального времени, продвинутыми инструментами построения графиков и различными типами ордеров, включая рыночные, лимитные и стоп-лимитные ордера.

### Маржинальная торговля
Для опытных трейдеров Coinone предлагает услуги маржинальной торговли. Эта функция позволяет пользователям заимствовать средства для торговли большими позициями, чем позволил бы их начальный капитал. Предлагаемое кредитное плечо варьируется в зависимости от торгуемой криптовалюты.

### Стейкинг
Coinone предоставляет услуги стейкинга для нескольких криптовалют, позволяя пользователям зарабатывать вознаграждения, участвуя в сети.

### Интеграция API
Coinone предлагает надежный API для разработчиков и алгоритмических трейдеров. API предоставляет доступ к различным функциям, включая рыночные данные в режиме реального времени, управление аккаунтом и исполнение ордеров.

### Безопасность
Coinone использует несколько уровней безопасности, включая двухфакторную аутентификацию (2FA), холодное хранение цифровых активов и регулярные аудиты безопасности.

### Соответствие нормативным требованиям
В соответствии с регуляторными требованиями Южной Кореи, Coinone требует прохождения процедур «Знай своего клиента» (KYC) и «Противодействие отмыванию денег» (AML).

## Техническая информация для алгоритмической торговли

### Конечные точки API
Coinone предоставляет несколько конечных точек API, которые могут использоваться для алгоритмической торговли:

- **Публичные конечные точки**: Эти конечные точки предоставляют доступ к рыночным данным без требования аутентификации.
 - `/ticker`: Возвращает текущую информацию о тикере.
 - `/orderbook`: Предоставляет данные ордербука.
 - `/trades`: Возвращает последние сделки.

- **Приватные конечные точки**: Эти конечные точки требуют аутентификации по ключу API.
 - `/balance`: Возвращает информацию о балансе пользователя.
 - `/order`: Позволяет размещать, отменять и проверять статус ордеров.
 - `/ledger`: Предоставляет историю транзакций для аккаунта.

## Дополнительные услуги

### Кошелек Coinone
Coinone предлагает безопасный цифровой кошелек для хранения различных криптовалют.

### Мобильное приложение
Coinone предоставляет мобильное приложение, доступное как на платформах iOS, так и Android.

### Coinone Pay
Coinone Pay — это платежный шлюз, который позволяет продавцам принимать платежи в криптовалюте.

## Заключение
Coinone — это комплексная платформа криптовалютной биржи, обслуживающая различные типы трейдеров, от новичков до продвинутых алгоритмических трейдеров.
