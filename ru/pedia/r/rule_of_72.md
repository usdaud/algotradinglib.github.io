# Правило 72

Правило 72 — это простая и полезная формула в финансах, которая оценивает количество лет, необходимых для удвоения стоимости инвестиции на основе фиксированной годовой нормы доходности. Это правило широко используется благодаря его простоте и примерной точности для понимания влияния сложных процентов во времени.

## Понимание Правила 72

Правило 72 гласит, что вы можете оценить время удвоения инвестиции, разделив 72 на годовую норму доходности (выраженную в процентах). Формула выглядит следующим образом:

```
Время удвоения (в годах) = 72 / Годовая норма доходности
```

Например, если у вас есть инвестиция, которая приносит 6% годовой процент, расчетное время для удвоения инвестиции составляет примерно:

```
72 / 6 = 12 лет
```

Аналогично, для инвестиции с 9% годовым доходом это займет:

```
72 / 9 = 8 лет
```

## Применение Правила 72

### Личные финансы

Физические лица могут использовать Правило 72 для быстрой оценки того, как долго будут расти их сбережения или инвестиции. Например, если у кого-то есть пенсионный счет со среднегодовым доходом 8%, они могут использовать Правило 72, чтобы примерно рассчитать, что их деньги удвоятся каждые 9 лет.

### Принятие инвестиционных решений

Финансовые консультанты и инвесторы часто используют Правило 72 для оценки различных инвестиционных возможностей. Установив, как долго инвестиция удвоится, они могут легче сравнивать привлекательность различных вариантов.

### Процентные ставки и кредиты

Правило 72 также можно применять для понимания влияния процентных ставок на кредиты. Например, если долг накапливает процентную ставку 12%, выдающийся остаток, вероятно, удвоится за 6 лет (72 разделено на 12).

## Ограничения и точность

Хотя Правило 72 — это полезное практическое правило, оно не совершенно точно. Формула наиболее точна для процентных ставок в диапазоне от 6% до 10%. Для очень низких или высоких процентных ставок оценка может быть менее точной. Поэтому Правило 72 лучше использовать для приблизительных расчетов и быстрых оценок, а не для точного финансового планирования.

## Математическая основа Правила 72

Правило 72 получено из формулы для сложных процентов:

```
A = P(1 + r/n)^(nt)
```

Здесь `A` — сумма денег, накопленная после `n` периодов; `P` — сумма основного капитала (первоначальная сумма денег); `r` — годовая процентная ставка; `n` — количество раз, когда проценты начисляются за год; и `t` — количество лет, на которые деньги инвестируются или заимствуются.

Чтобы узнать, когда инвестиция удвоится (т.е. когда `A = 2P`), мы упрощаем формулу с непрерывным начислением процентов (`n` приближается к бесконечности):

```
2 = (1 + r)^t
ln(2) = t*ln(1 + r)
```

Для малых `r`, `ln(1 + r) ≈ r`, поэтому:

```
ln(2) ≈ t*r
t ≈ ln(2) / r
t ≈ 0,693 / r
```

Поскольку 0,693 примерно равна 0,72 (и для упрощения мы используем 72 вместо более точных значений типа 69 или 70), Правило 72 дает нам:

```
t ≈ 72 / (r * 100)
```

Где `r` выражается в процентах.

## Вариации и расширения

### Правило 70 и Правило 69,3

Варианты Правила 72, такие как Правило 70 и Правило 69,3, предлагают небольшие корректировки, чтобы учесть различные сценарии. Эти варианты иногда обеспечивают более точные результаты для определенных диапазонов процентных ставок или частот начисления процентов.

### Корректировки для различных периодов начисления

Правило 72 в целом предполагает годовое начисление. Для других периодов начисления формула все еще может быть приспособлена, но с другим делителем. Например, если проценты начисляются ежемесячно, вы можете использовать 78 вместо 72 для более точной оценки.

### За пределами удвоения

Правило 72 можно изменить для оценки утроения или более высоких кратных путем корректировки числителя. Для утроения инвестиции вы бы использовали Правило 114 (поскольку `ln(3)` приближается к `1,1`). Для учетверения используется Правило 144.

## Практические примеры на финансовых рынках

### Инвестиции в фондовый рынок

Предположим, вы инвестируете в диверсифицированный портфель акций со средним годовым доходом 7%. Используя Правило 72:

```
72 / 7 = примерно 10,3 года
```

Это означает, что ваша инвестиция, вероятно, удвоится чуть более чем за 10 лет.

### Ценные бумаги с фиксированным доходом

Если вы инвестируете в облигацию, которая приносит 4% годовых, Правило 72 предсказывает:

```
72 / 4 = 18 лет
```

Таким образом, требуется 18 лет, чтобы ваш основной капитал удвоился.

## Важные соображения

### Корректировка инфляции

При использовании Правила 72 важно рассмотреть влияние инфляции на реальную стоимость денег. Инвестиция может удвоиться в номинальном выражении, но если инфляция была высокой, реальная покупательная способность этого удвоившегося количества может быть значительно меньше.

### Компромисс между риском и доходностью

Правило 72 не учитывает риск. Более высокие доходы часто сопровождаются более высоким риском, и хотя Правило помогает в понимании потенциального роста, важно сбалансировать смету доходности с связанными рисками.

### Влияние налогов

Налоги могут значительно повлиять на рост инвестиций. Расчет Правила 72 предполагает рост без налогов, что редко бывает в реальных сценариях.

### Неежегодное начисление

Если проценты начисляются чаще, чем ежегодно (например, ежемесячно или ежеквартально), расчетное время удвоения по Правилу 72 может быть менее точным. Различные корректировки или альтернативные правила могут быть необходимы для большей точности.

## Заключение

Правило 72 — это простое, но мощное практическое правило, помогающее при быстром расчете времени удвоения инвестиций. Его широкое применение в личных финансах, анализе инвестиций и понимании влияния кредитов делает его незаменимым инструментом как для новичков, так и для профессионалов в финансовом секторе. Однако пользователи должны быть осведомлены о его ограничениях и рассмотреть корректировки для большей точности в различных финансовых сценариях.
