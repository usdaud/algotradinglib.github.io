# 2-Period RSI

Индекс относительной силы (RSI) — известный осциллятор импульса, разработанный Дж. Уэллсом Уайлдером, который измеряет скорость и величину изменения цены. RSI колеблется от 0 до 100 и обычно используется для выявления состояний перекупленности или перепроданности на рынке. Хотя стандартный расчет RSI основан на 14 периодах, 2-периодный RSI — это вариация для более коротких таймфреймов, дающая более чувствительные и частые сигналы. Такой подход полезен активным трейдерам, работающим на коротком горизонте.

## Понимание RSI

Прежде чем переходить к 2-периодному RSI, важно понимать базовую формулу традиционного RSI. Он рассчитывает отношение средних ростов к средним падениям за выбранный период:

```
RS = (Average Gain over N periods) / (Average Loss over N periods)
RSI = 100 - [100 / (1 + RS)]
```

В этих формулах RS означает относительную силу (Relative Strength), Average Gain — среднее значение всех ростов за выбранный период, а Average Loss — среднее значение всех падений.

## Расчет 2-периодного RSI

Для 2-периодного RSI период усреднения сокращается до двух периодов вместо традиционных 14. Это делает индикатор более быстрым и отзывчивым, позволяя эффективнее ловить краткосрочные движения цены.

Порядок расчета:

1. **Рассчитать средние рост и падение:**
 Для каждого из последних двух периодов определить, выросла или упала цена и на сколько.

2. **Определить RS:**
 Разделить средний рост на среднее падение, чтобы получить RS.

3. **Подставить RS в формулу RSI:**
 Использовать RS в формуле RSI: `RSI = 100 - [100 / (1 + RS)]`.

### Пример расчета

Если трейдер рассматривает цену акции за последние три дня:

- День 1: цена выросла с $50 до $52 (Gain = $2)
- День 2: цена снизилась с $52 до $51 (Loss = $1)
- День 3: цена выросла с $51 до $53 (Gain = $2)

Average Gain = (2 + 2) / 2 = 2\
Average Loss = 1 / 2 = 0.5\
RS = 2 / 0.5 = 4\
RSI = 100 - [100 / (1 + 4)] = 100 - 20 = 80

## Торговля с 2-периодным RSI

### Сигналы покупки и продажи

Главное преимущество 2-периодного RSI — более частые сигналы по сравнению с 14-периодным RSI, что делает его подходящим для дневной или свинг-торговли. Типичные сигналы:

- **Сигнал покупки:** когда 2-периодный RSI опускается ниже 10, это может указывать на перепроданность и создавать возможность для покупки.
- **Сигнал продажи:** когда 2-периодный RSI поднимается выше 90, это может указывать на перекупленность и создавать возможность для продажи.

### Комбинация с другими индикаторами

Для повышения надежности трейдеры часто комбинируют 2-периодный RSI с другими индикаторами, такими как скользящие средние, MACD или уровни поддержки/сопротивления. Использование нескольких индикаторов помогает подтверждать сигналы и снижать число ложных срабатываний.

### Бэктестинг и оптимизация

Перед применением стратегии с 2-периодным RSI важно провести бэктестинг и оценить ее эффективность в разных рыночных условиях. На основе исторических данных можно оптимизировать период и пороговые значения.

## Преимущества и ограничения

### Преимущества

- **Отзывчивость:** быстро реагирует на движения рынка, что делает индикатор пригодным для краткосрочной торговли.
- **Простота:** расчет прост и легко реализуется на торговых платформах.

### Ограничения

- **Больше ложных сигналов:** из-за высокой чувствительности индикатор чаще генерирует ложные сигналы.
- **Рыночный шум:** в условиях высокой волатильности 2-периодный RSI может реагировать на шум, а не на значимые тренды.

## Практическое применение

Многие торговые платформы и брокеры предлагают встроенные инструменты для расчета RSI, включая 2-периодную версию. Среди популярных платформ — MetaTrader, TradingView и ThinkOrSwim.

### Реализация в MetaTrader

Для создания пользовательского индикатора на MetaTrader трейдеры могут использовать язык MQL4/MQL5. Ниже приведен базовый каркас реализации:

```mql4
int OnInit()
  {
   IndicatorBuffers(1);
   SetIndexBuffer(0,RSIBuffer);
   SetIndexStyle(0,DRAW_LINE);
   string short_name = "2-Period RSI";
   IndicatorShortName(short_name);
   IndicatorDigits(Digits);
   return(INIT_SUCCEEDED);
  }
int OnCalculateconst int rates_total,
              const int prev_calculated,
              const datetime &time[],
              const double &open[],
              const double &high[],
              const double &low[],
              const double &close[],
              const long &tick_volume[],
              const long &volume[],
              const int &spread[])
  {
   int limit = rates_total-prev_calculated;
   double gain, loss;
   for(int i=rates_total-limit; i>=0; i--)
     {
      gain = 0.0;
      loss = 0.0;
      for(int j=1; j<=2; j++)
        {
         double diff = close[i-j+1] - close[i-j];
         if (diff > 0) gain += diff;
         else loss -= diff;
        }
      gain /= 2.0;
      loss /= 2.0;
      if (loss == 0.0) RSIBuffer[i] = 100;
      else if (gain == 0.0) RSIBuffer[i] = 0;
      else
        {
         double RS = gain/loss;
         RSIBuffer[i] = 100.0 - (100.0 / (1.0 + RS));
        }
     }
   return(rates_total);
  }
```

### Реализация в TradingView

На TradingView можно использовать Pine Script для создания пользовательского 2-периодного RSI:

```pinescript
//@version=4
study(title="2-Period RSI", shorttitle="2-RSI", overlay=false)
rsiLength = 2
source = close
rsiValue = rsi(source, rsiLength)
plot(rsiValue, title="2-Period RSI", color=color.blue)
hline(90, "Overbought", color=color.red)
hline(10, "Oversold", color=color.green)
```

### Реализация в ThinkOrSwim

ThinkOrSwim использует язык ThinkScript. Ниже пример 2-периодного RSI:

```thinkscript
declare lower;
input length = 2;
def NetChgAvg = WildersAverage(close - close[1], length);
def TotChgAvg = WildersAverage(Absvalue(close - close[1]), length);
def ChgRatio = if TotChgAvg != 0 then NetChgAvg / TotChgAvg else 0;
plot RSI = 50 * (ChgRatio + 1);
RSI.SetDefaultColor(GetColor(1));
plot Overbought = 90;
Overbought.SetDefaultColor(GetColor(5));
plot Oversold = 10;
Oversold.SetDefaultColor(GetColor(5));
```

## Заключение

2-периодный RSI — мощный инструмент для краткосрочных трейдеров, которые хотят улавливать быстрые движения рынка. Он сочетает отзывчивость и простоту, но требует осторожности из-за повышенной чувствительности и необходимости фильтрации ложных сигналов. При грамотном использовании и подтверждении другими индикаторами 2-периодный RSI может существенно усилить торговый план.
