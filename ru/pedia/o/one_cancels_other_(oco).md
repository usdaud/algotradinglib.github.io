# Один отменяет другой (OCO)

Ордер "один отменяет другой" — это связанная пара ордеров, где исполнение одного ордера автоматически отменяет другой. Обычно используется для одновременной установки цели по прибыли и стоп-лосса.

## Как это работает
- Два ордера связаны: один для фиксации прибыли и один для контроля рисков.
- Когда один ордер исполняется, другой отменяется.
- Трейдер отслеживает только одну комбинированную инструкцию.

## Пример
Трейдер покупает акцию по 50 и устанавливает OCO с лимитным ордером на продажу по 55 и стопом на 47. Если цена достигает 55, стоп отменяется. Если цена достигает 47, целевой ордер отменяется.

## Риски
Ордера OCO уменьшают ручные ошибки, но все равно сталкиваются с гэпами и проскальзыванием. Если цена проскакивает через уровень стопа, исполнение может быть хуже ожидаемого.
