# Zerodha

## Обзор
Zerodha - крупнейший розничный фондовый брокер Индии, предлагающий широкий спектр финансовых услуг, включая торговлю акциями, деривативами, товарами, паевые фонды и облигации. Компания известна моделью дисконтного брокера, продвинутыми торговыми платформами и обширными образовательными ресурсами. Zerodha ориентирована на частных инвесторов, трейдеров и институциональных клиентов.

## Особенности
1. **Широкий набор инструментов**: доступ к акциям, деривативам, товарам, паевым фондам и облигациям.
2. **Дисконтный брокер**: низкая стоимость сделок с фиксированной комиссией за сделку, что делает компанию одним из самых экономичных брокеров в Индии.
3. **Продвинутые торговые платформы**: несколько платформ, включая Kite, Coin, Console и Kite Mobile, с расширенными графиками и данными в реальном времени.
4. **Образовательные ресурсы**: обширные материалы, включая вебинары, руководства, статьи и рыночную аналитику через Zerodha Varsity.
5. **Рыночные исследования и аналитика**: доступ к комплексным отчетам, новостям рынка и продвинутым инструментам анализа.
6. **Доступ к API**: надежные API для автоматизированной торговли, интеграций и разработки сторонних торговых приложений.
7. **Управление портфелем**: инструменты для эффективного отслеживания и управления инвестиционными портфелями.
8. **Мобильная торговля**: приложения для iOS и Android, позволяющие торговать на ходу.
9. **Инвестиции в паевые фонды**: платформа для инвестирования в прямые паевые фонды без комиссии.
10. **Безопасность и регулирование**: Zerodha регулируется ведущими финансовыми органами Индии, что обеспечивает безопасную торговую среду.

## Ключевые компоненты
- **Kite**: мощная торговая платформа с расширенными графиками, данными в реальном времени и набором торговых инструментов.
- **Kite Mobile**: мобильное приложение для торговли, управления счетом и рыночных исследований на ходу.
- **Coin**: платформа для инвестиций в прямые паевые фонды.
- **Console**: бэк-офисная платформа для отслеживания инвестиций и формирования отчетов.
- **Образовательные ресурсы (Varsity)**: библиотека материалов, включая вебинары, руководства и статьи.
- **Доступ к API**: API для автоматизированной торговли и кастомных интеграций.

## Интеграции
Zerodha интегрируется с различными финансовыми платформами и инструментами для расширения торговых услуг. Среди заметных интеграций:

- **Поставщики рыночных данных**: интеграции с ведущими поставщиками рыночных данных для информации в реальном времени и исторических данных.
- **Сторонние платформы**: интеграции со сторонними торговыми платформами и инструментами.
- **Доступ к API**: предоставляет API для кастомных интеграций и решений автоматизированной торговли.
- **Исследовательские инструменты**: доступ к сторонним инструментам анализа и исследований.

## Сценарии использования
- **Частные инвесторы**: используют для управления инвестициями, доступа к исследованиям и торговли на различных рынках.
- **Активные трейдеры**: получают мощные торговые инструменты, данные рынка в реальном времени и продвинутую аналитику.
- **Новые трейдеры**: поддержка через образовательные ресурсы и удобную платформу.
- **Инвесторы, чувствительные к затратам**: модель дисконтного брокера помогает снижать торговые издержки.
- **Алгоритмические трейдеры**: поддержка разработки и запуска стратегий через API и торговые платформы.
- **Портфельные управляющие**: помощь в отслеживании и управлении портфелями с помощью отчетов и инструментов.

## Регулирование
Zerodha регулируется рядом ведущих финансовых органов Индии, обеспечивая безопасную и соответствующую требованиям торговую среду:
- Securities and Exchange Board of India (SEBI)
- National Stock Exchange (NSE)
- Bombay Stock Exchange (BSE)
- Multi Commodity Exchange (MCX)
