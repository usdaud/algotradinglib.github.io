# Questrade

## Обзор
Questrade — это ведущий онлайн-брокер в Канаде, известный своей недорогой торговлей, комплексными торговыми платформами и широким спектром инвестиционных продуктов. Платформа обслуживает индивидуальных инвесторов и трейдеров, предлагая услуги по акциям, ETF, опционам, взаимным фондам и многому другому. Questrade стремится предоставлять инновационные технологии, ценные образовательные ресурсы и исключительное обслуживание клиентов.

## Особенности
1. **Недорогая торговля**: Конкурентные цены с низкими комиссиями и прозрачной структурой сборов.
2. **Широкий спектр инструментов**: Доступ к акциям, ETF, опционам, взаимным фондам, GIC и облигациям.
3. **Продвинутые торговые платформы**: Предлагает несколько торговых платформ, включая Questrade Trading, Questrade IQ Edge и мобильные приложения с расширенными инструментами построения графиков и данными в реальном времени.
4. **Исследования и анализ**: Комплексные инструменты для исследований, включая скринеры акций, отчеты аналитиков, рыночные новости и технический анализ.
5. **Образовательные ресурсы**: Обширные образовательные материалы, включая вебинары, учебные пособия, статьи и рыночный анализ, чтобы помочь трейдерам улучшить свои навыки.
6. **Управление портфелем**: Инструменты для эффективного отслеживания и управления инвестиционными портфелями.
7. **Типы счетов**: Различные типы счетов, включая RRSP, TFSA, маржинальные счета и другие.
8. **Автоматизированное инвестирование**: Робо-консультационные услуги через Questwealth Portfolios для автоматизированного недорогого управления инвестициями.
9. **Мобильная торговля**: Мобильные приложения для устройств iOS и Android, позволяющие торговать на ходу.
10. **Безопасность и регулирование**: Questrade регулируется ведущими финансовыми органами Канады, обеспечивая безопасную торговую среду.

## Ключевые компоненты
- **Questrade Trading**: Удобная торговая платформа, предлагающая расширенные графики, данные в реальном времени и набор торговых инструментов.
- **Questrade IQ Edge**: Продвинутая торговая платформа, разработанная для активных трейдеров, предоставляющая надежные инструменты построения графиков, технический анализ и настраиваемые макеты.
- **Мобильные торговые приложения**: Мобильные приложения для iOS и Android с полным функционалом торговли.
- **Инструменты для исследований и анализа**: Комплексные инструменты для рыночных исследований, анализа акций и инвестиционных инсайтов.
- **Образовательные ресурсы**: Библиотека образовательного контента, включая вебинары, учебные пособия и статьи.
- **Questwealth Portfolios**: Служба автоматизированного управления инвестициями, использующая диверсифицированные портфели ETF.
- **Инструменты управления портфелем**: Инструменты для эффективного отслеживания и управления инвестиционными портфелями.

## Интеграции
Questrade интегрируется с различными финансовыми платформами и инструментами для расширения своих торговых услуг. Некоторые заметные интеграции включают:

- **Поставщики рыночных данных**: Интеграция с ведущими поставщиками рыночных данных для получения данных в реальном времени и исторических данных.
- **TradingView**: Интеграция с TradingView для расширенного построения графиков и технического анализа.
- **Сторонние исследования**: Доступ к сторонним инструментам исследований и анализа.
- **Инструменты управления портфелем**: Интеграция с инструментами управления портфелем и отслеживания.
- **API**: Предоставляет API для пользовательских интеграций и решений для автоматизированной торговли.

## Варианты использования
- **Индивидуальные инвесторы**: Используется индивидуальными инвесторами для управления своими инвестициями, доступа к исследованиям и торговли на различных рынках.
- **Активные трейдеры**: Предоставляет активным трейдерам мощные торговые инструменты, рыночные данные в реальном времени и расширенный анализ.
- **Клиенты робо-консультантов**: Предлагает автоматизированное управление инвестициями через Questwealth Portfolios для инвесторов с пассивным подходом.
- **Образовательные цели**: Поддерживает новых и опытных трейдеров обширными образовательными ресурсами и рыночным анализом.
- **Управляющие портфелями**: Помогает управляющим портфелями отслеживать и управлять своими инвестиционными портфелями с помощью комплексных инструментов и отчетов.

## Регулирование
Questrade регулируется несколькими ведущими финансовыми органами в Канаде, обеспечивая безопасную и соответствующую нормам торговую среду:
- Организация по регулированию инвестиционной индустрии Канады (IIROC)
- Канадский фонд защиты инвесторов (CIPF)
