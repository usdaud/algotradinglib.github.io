# Международный номер банковского счета (IBAN)

Международный номер банковского счета (IBAN) — это международно стандартизированная система идентификации банковских счетов через национальные границы. Её основная цель — сделать трансграничные платежи более эффективными и безопасными. IBAN первоначально был разработан для упрощения платежей внутри Европейского союза, но с тех пор был принят другими странами и регионами по всему миру. В данном документе представлено подробное объяснение того, что такое IBAN, его структура, статус глобального принятия, преимущества и сравнение с другими международными платежными идентификаторами.

## Что такое IBAN?

IBAN — это уникальный код, который идентифицирует конкретный банковский счет в конкретном финансовом учреждении в конкретной стране. Он предназначен для упрощения и ускорения международных банковских операций, предоставляя единый формат, который может быть легко проверен и обработан банками и другими финансовыми учреждениями.

## Структура IBAN

IBAN состоит из до 34 буквенно-цифровых символов, которые включают следующие компоненты:

1. **Код страны**: Первые две буквы представляют страну, в которой находится банковский счет. Этот код основан на стандарте ISO 3166-1 alpha-2. Например, "GB" для Великобритании, "DE" для Германии или "FR" для Франции.

2. **Контрольные цифры**: Следующие две цифры обеспечивают метод проверки IBAN для предотвращения ошибок при переписывании. Эти цифры вычисляются с использованием алгоритма MOD-97-10.

3. **Базовый номер банковского счета (BBAN)**: Оставшиеся символы формируют базовый номер банковского счета, который варьируется по длине и формату от страны к стране. BBAN включает код банка, код филиала и сам номер счета.

### Пример IBAN

IBAN для банковского счета в Германии может выглядеть так:
```
DE89 3704 0044 0532 0130 00
```

Здесь "DE" обозначает Германию, "89" — контрольные цифры, а "3704 0044 0532 0130 00" — BBAN.

## Глобальное принятие

Использование IBAN стало обязательным в Европейском союзе для всех трансграничных транзакций внутри ЕС и Европейской экономической зоны (ЕЭЗ). Многие другие страны мира также приняли эту систему для упрощения международных транзакций. Некоторые из регионов и стран, принявших IBAN:

- Европейский союз (27 государств-членов)
- Европейская ассоциация свободной торговли (Исландия, Лихтенштейн, Норвегия, Швейцария)
- Другие страны (Албания, Алжир, Андорра, Азербайджан, Бахрейн, Босния и Герцеговина, Бразилия, Коста-Рика, Грузия, Иран, Израиль, Иордания, Казахстан, Кувейт, Ливан, Мавритания, Маврикий, Молдова, Монако, Черногория, Пакистан, Палестина, Катар, Сан-Марино, Саудовская Аравия, Сербия, Тунис, Турция, Объединенные Арабские Эмираты)

Около 70 стран приняли систему IBAN.

## Преимущества использования IBAN

Использование IBAN предоставляет несколько преимуществ для международных финансовых транзакций:

1. **Эффективность**: Стандартизируя формат номеров банковских счетов, IBAN сокращает время и усилия, необходимые для обработки международных транзакций.
2. **Сокращение ошибок**: Контрольные цифры помогают предотвратить ошибки при вводе данных, обеспечивая перевод средств на правильный счет.
3. **Снижение затрат**: Более быстрое время обработки и сниженный уровень ошибок помогают снизить расходы, связанные с международными платежами.
4. **Улучшенная автоматизация**: Единообразный формат позволяет повысить автоматизацию систем обработки платежей, дополнительно повышая эффективность и сокращая человеческое вмешательство.

## Сравнение IBAN с другими платежными идентификаторами

IBAN — лишь одна из нескольких систем, предназначенных для упрощения международных платежей. Вот некоторые распространенные альтернативы:

- **Код SWIFT/BIC**: Общество всемирных межбанковских финансовых телекоммуникаций (SWIFT) присваивает коды бизнес-идентификаторов (BIC) финансовым учреждениям. Эти коды используются наряду с IBAN для международных переводов.

- **Маршрутный номер ABA**: Используется преимущественно в США, маршрутный номер Американской банковской ассоциации (ABA) идентифицирует финансовые учреждения, но не стандартизирован для международного использования.

- **UID CHIPS**: Универсальный идентификатор (UID) системы межбанковских платежей расчетной палаты (CHIPS) используется в США для крупных платежей, часто внутри одной страны.

Хотя IBAN обеспечивает более стандартизированный формат для трансграничных операций, коды SWIFT/BIC и другие идентификаторы по-прежнему широко используются совместно с IBAN для обеспечения точной маршрутизации международных платежей.

### Пример работы IBAN

Рассмотрим пример международного платежа из банка во Франции в банк в Германии.

1. Банк отправителя во Франции запросит IBAN счета получателя в Германии.
2. Банк отправителя проверит действительность IBAN с помощью контрольных цифр.
3. После проверки банк отправителя инициирует перевод через сеть SWIFT, включая как IBAN получателя, так и код SWIFT/BIC банка получателя.
4. Банк получателя в Германии получит платежное поручение и зачислит средства на указанный счет на основе IBAN.

Этот процесс обеспечивает точное и эффективное направление платежа предполагаемому получателю.

## Заключение

Международный номер банковского счета (IBAN) является важнейшим компонентом современного международного банковского дела, предоставляя стандартизированный и надежный метод идентификации банковских счетов через национальные границы. Его принятие значительно повысило эффективность, точность и экономичность трансграничных финансовых транзакций. По мере продолжения расширения глобальной торговли использование IBAN, вероятно, станет ещё более распространенным, что ещё больше упростит международные платежи.
