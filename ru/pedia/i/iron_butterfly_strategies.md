# Стратегии железной бабочки

## Введение
Железная бабочка, часто называемая просто "Iron Fly", является популярной стратегией торговли опционами, используемой трейдерами для извлечения выгоды из узкого диапазона цены базового актива. Эта стратегия представляет собой комбинацию бычьего спреда пут и медвежьего спреда колл. Железная бабочка является рыночно-нейтральной стратегией, что означает, что она может быть прибыльной в сценариях, где базовая акция не движется значительно.

## Компоненты железной бабочки
Стратегия железной бабочки состоит из четырех опционов:
1. Купить один опцион пут вне денег (более низкая цена исполнения).
2. Продать один опцион пут при деньгах (средняя цена исполнения).
3. Продать один опцион колл при деньгах (средняя цена исполнения).
4. Купить один опцион колл вне денег (более высокая цена исполнения).

Опционы пут и колл при деньгах (ATM) имеют одинаковую цену исполнения, которая обычно близка к текущей цене базового актива. Эти средние страйковые опционы являются ядром железной бабочки, поскольку они определяют диапазон, в пределах которого стратегия приносит прибыль.

## Построение железной бабочки
Для построения железной бабочки выполните следующие шаги:
1. **Выберите базовый актив**: Выберите акцию или индекс, который, по вашему мнению, не будет демонстрировать больших движений цены в течение срока действия опционов.
2. **Определите дату экспирации**: Выберите дату истечения срока действия опционных контрактов на основе вашего прогноза для базового актива.
3. **Выберите среднюю цену исполнения**: Выберите цену исполнения при деньгах, которая близка к текущей рыночной цене базового актива.
4. **Установите ширину крыльев**: Определите цены исполнения для опционов вне денег (как колл, так и пут), которые будут формировать "крылья" бабочки.

Например, если акция торгуется по $50:
- Купить один опцион пут 45 (вне денег).
- Продать один опцион пут 50 (при деньгах).
- Продать один опцион колл 50 (при деньгах).
- Купить один опцион колл 55 (вне денег).

## Профиль риска и вознаграждения
- **Максимальная прибыль**: Возникает, когда цена базового актива находится на среднем страйке (т.е. на страйке при деньгах) при экспирации. Прибыль ограничена чистым кредитом, полученным при создании железной бабочки.
- **Максимальный убыток**: Возникает, когда цена базового актива при экспирации находится ниже более низкой цены исполнения пут или выше более высокой цены исполнения колл. Убыток ограничен разницей между нижними/верхними страйками и средним страйком минус полученный чистый кредит.

Точки безубыточности для железной бабочки рассчитываются следующим образом:
- Нижняя точка безубыточности: Средняя цена исполнения - Полученный чистый кредит.
- Верхняя точка безубыточности: Средняя цена исполнения + Полученный чистый кредит.

## Пример
Рассмотрим железную бабочку со следующими опционами:
- Купить 1 пут XYZ Dec 45 по $1
- Продать 1 пут XYZ Dec 50 по $3
- Продать 1 колл XYZ Dec 50 по $3
- Купить 1 колл XYZ Dec 55 по $1

Полученный чистый кредит составляет ($3 + $3) - ($1 + $1) = $4.

- **Максимальная прибыль**: Достигается, если XYZ закрывается на $50 при экспирации. Прибыль = $4.
- **Максимальный убыток**: Возникает ниже $45 или выше $55. Убыток = ($5 - $4) = $1.

## Корректировки и вариации
Иногда трейдерам необходимо корректировать свои позиции железной бабочки для управления риском или повышения прибыльности. Возможные корректировки включают:
- **Ролл**: Перемещение всей позиции на другую экспирацию или цены исполнения.
- **Раскрытие**: Закрытие одной или нескольких ног железной бабочки для преобразования её в другую стратегию, такую как вертикальный спред или железный кондор.

## Железная бабочка против железного кондора
Обе стратегии железной бабочки и железного кондора похожи; однако есть ключевые различия:
- **Железная бабочка**: Использует опционы при деньгах для коротких страйков, что приводит к более высокому чистому кредиту, но более узкому диапазону прибыли.
- **Железный кондор**: Использует опционы вне денег для коротких страйков, что дает более низкий чистый кредит, но более широкий диапазон прибыли.

## Налоговые последствия
Стратегия железной бабочки может иметь налоговые последствия, поскольку торговля опционами может облагаться правилами налога на краткосрочный прирост капитала в большинстве юрисдикций. Трейдерам крайне важно консультироваться с налоговыми специалистами для понимания конкретного налогового воздействия на их торговую деятельность.

## Заключение
Железная бабочка является универсальной и популярной стратегией для трейдеров, стремящихся извлечь прибыль из нейтрального взгляда на рынок. Понимая компоненты, построение, профили риска и вознаграждения, а также потенциальные корректировки, трейдеры могут эффективно использовать эту стратегию для улучшения своих торговых портфелей. Однако, как и любая торговая стратегия, она сопряжена с присущими рисками и требует тщательного анализа и управления рисками.
