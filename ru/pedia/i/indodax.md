# Indodax

Indodax — это крупнейшая и наиболее комплексная биржа цифровых активов, базирующаяся в Индонезии. Первоначально известная как Bitcoin.co.id, платформа была переименована в Indodax (Indonesia Digital Asset Exchange) в марте 2018 года. Indodax предоставляет безопасную и удобную площадку для пользователей для покупки и продажи широкого спектра криптовалют. Это важный игрок на рынке криптовалют как для индонезийских граждан, так и для других пользователей по всему миру, с более чем 5 миллионами зарегистрированных участников по состоянию на 2023 год.

Платформа поддерживает торговлю различными криптовалютами, включая Bitcoin (BTC), Ethereum (ETH), Ripple (XRP) и большим количеством альткоинов. Indodax также облегчает торговлю с индонезийской рупией (IDR), что упрощает местным пользователям покупку и продажу криптовалют с использованием их национальной валюты.

## Функции платформы

### Удобный интерфейс
Indodax разработан так, чтобы быть удобным как для начинающих, так и для опытных трейдеров. Интерфейс интуитивно понятен, что облегчает пользователям навигацию между различными функциями, такими как управление кошельками, торговые терминалы и учетные записи пользователей.

### Меры безопасности
Безопасность является первостепенной заботой для любой биржи цифровых активов. Indodax внедряет продвинутые меры безопасности, такие как двухфакторная аутентификация (2FA), зашифрованная связь и холодное хранилище для большей части своих цифровых активов. Регулярные проверки безопасности проводятся для обеспечения целостности платформы.

### Разнообразный портфель активов
Indodax поддерживает длинный список криптовалют, предоставляя пользователям возможность диверсифицировать свои портфели. Пользователи могут торговать популярными криптовалютами, такими как Bitcoin и Ethereum, а также различными менее известными альткоинами.

### Торговля фиат-крипто и крипто-крипто
Одной из ключевых функций Indodax является возможность торговать напрямую между индонезийской рупией (IDR) и различными криптовалютами. Это устраняет необходимость для пользователей конвертировать свою местную валюту в USD или другой стейблкоин перед торговлей. В дополнение к торговым парам фиат-крипто, Indodax также поддерживает многочисленные торговые пары крипто-крипто.

### Ликвидность
Indodax может похвастаться высокой ликвидностью, что обеспечивает плавное и быстрое выполнение транзакций. Это имеет решающее значение для трейдеров, которым необходимо быстро входить и выходить из позиций, чтобы воспользоваться рыночными возможностями.

## API для алгоритмической торговли

Indodax предоставляет доступ к API для разработчиков и алгоритмических трейдеров, которые хотят автоматизировать свои торговые стратегии. API позволяет пользователям выполнять сделки, получать рыночные данные в реальном времени, управлять своими счетами и многое другое. Ниже приведены некоторые важные детали об API:

### Конечные точки API
- **Рыночные данные:** Получение данных о ценах в реальном времени и исторических данных для поддерживаемых торговых пар.
- **Торговля:** Размещение и отмена ордеров, проверка статуса ордера.
- **Управление счетом:** Доступ к балансам счета, информация о депозитах и снятии средств.
- **Публичные конечные точки:** Получение сводок рыночных данных, включая информацию о тикере, объем торгов и последние сделки.
- **Приватные конечные точки:** Требует аутентификации с помощью API-ключа для получения и управления данными пользователя.

### Аутентификация
API использует ключи API для аутентификации. Пользователи могут генерировать свои API-ключи через настройки своего аккаунта Indodax. Каждый API-ключ поставляется с разрешениями и ограничениями, которые могут быть настроены для различных вариантов использования.

### Ограничение частоты запросов
Для обеспечения справедливого использования и избежания перегрузки системы, Indodax внедряет ограничение частоты запросов API. Пользователям рекомендуется обращаться к документации API для определения конкретных ограничений частоты и убедиться, что их приложения соблюдают их.

### Документация
Комплексная документация по API доступна на онлайн-платформе Indodax для помощи разработчикам в интеграции API с их приложениями. Документация включает подробные описания всех доступных конечных точек, параметров и примеров фрагментов кода.


## Способы оплаты

Indodax поддерживает различные способы оплаты для депозита и снятия индонезийской рупии (IDR):
- **Банковский перевод:** Поддерживаются прямые банковские переводы из крупных индонезийских банков. Это наиболее распространенный метод внесения и снятия средств.
- **Электронные кошельки:** Интеграция с популярными услугами электронных кошельков в Индонезии позволяет пользователям легко вносить и снимать средства.
- **Кредитная/дебетовая карта:** Некоторые пользователи могут иметь возможность использовать свои кредитные или дебетовые карты для транзакций, хотя это может сопровождаться дополнительными комиссиями.

## Комиссии

Indodax использует структуру комиссий, которая является конкурентоспособной и прозрачной:

### Торговые комиссии
Indodax использует модель комиссий мейкер-тейкер:
- **Комиссии мейкера:** Обычно ниже, так как они добавляют ликвидность в книгу ордеров.
- **Комиссии тейкера:** Немного выше, так как они удаляют ликвидность из книги ордеров.
Процентные ставки комиссий могут варьироваться, но обычно составляют около 0,15% для мейкеров и 0,25% для тейкеров.

### Комиссии за снятие средств
Комиссии за снятие средств варьируются в зависимости от криптовалюты или метода снятия. Комиссии за снятие IDR через банковский перевод фиксированы, в то время как снятие криптовалюты зависит от сетевых комиссий за транзакцию.

## Соответствие нормативным требованиям

Работая в Индонезии, Indodax соблюдает местные нормативные требования для обеспечения безопасности и законности своей деятельности. Платформа соответствует правилам по борьбе с отмыванием денег (AML) и знанию клиента (KYC), требуя от пользователей пройти верификацию личности перед доступом к полным функциям биржи.

## Образовательные ресурсы

Чтобы помочь как новым, так и опытным трейдерам, Indodax предоставляет множество образовательных ресурсов:
- **Блог и статьи:** Регулярные обновления о рыночных трендах, новых функциях и образовательном контенте.
- **Учебные пособия:** Пошаговые руководства о том, как начать торговлю, использовать продвинутые функции и внедрять меры безопасности.
- **Вебинары и мероприятия:** Живые сессии, где пользователи могут учиться непосредственно у экспертов и задавать вопросы в режиме реального времени.
