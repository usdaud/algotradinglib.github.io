# Deltix

**Примечание:** Deltix была приобретена EPAM Systems в июне 2021 года и теперь работает как часть EPAM.

## Обзор
Deltix — ведущий поставщик программного обеспечения и услуг для количественных исследований, алгоритмической и систематической торговли. Их решения разработаны для поддержки всего жизненного цикла стратегий количественной торговли, от исследований и разработки до исполнения и развертывания.

## Возможности
1. **Количественные исследования**: Инструменты для разработки, тестирования и оптимизации стратегий количественной торговли.
2. **Алгоритмическая торговля**: Комплексная инфраструктура для исполнения алгоритмических торговых стратегий по нескольким классам активов.
3. **Управление данными**: Возможности для обработки и анализа больших массивов данных из различных источников.
4. **Бэктестинг**: Высокопроизводительный движок бэктестинга для моделирования торговых стратегий с использованием исторических данных.
5. **Аналитика в реальном времени**: Обработка данных и аналитика в реальном времени для принятия обоснованных торговых решений.
6. **Управление рисками**: Продвинутые инструменты управления рисками для мониторинга и контроля торговых рисков.
7. **Исполнение ордеров**: Исполнение ордеров с низкой задержкой с подключением к крупным биржам и поставщикам ликвидности.
8. **Облако и локальное развертывание**: Варианты развертывания включают как облачные, так и локальные решения для удовлетворения различных операционных потребностей.

## Ключевые компоненты
- **QuantOffice**: Мощная среда разработки для создания, бэктестинга и оптимизации стратегий количественной торговли.
- **QuantServer**: Движок обработки событий в реальном времени, поддерживающий исполнение торговых стратегий с низкой задержкой.
- **QuantAnalyzer**: Инструмент для проведения углубленного анализа торговых стратегий, включая риск и показатели эффективности.
- **TimeBase**: Высокопроизводительная база данных временных рядов, разработанная для хранения и запроса больших объемов рыночных данных.
- **QuantWeb**: Веб-интерфейс для мониторинга и контроля торговых стратегий в реальном времени.
- **Сервер исполнения**: Управляет исполнением ордеров на нескольких рынках и через брокеров.

## Интеграции
Deltix интегрируется с различными поставщиками данных, биржами и торговыми платформами для обеспечения бесшовного торгового опыта. Некоторые заметные интеграции включают:

- **Поставщики рыночных данных**: Интеграция с ведущими поставщиками рыночных данных для получения данных в реальном времени и исторических данных.
- **Биржи и брокеры**: Подключение к крупным глобальным биржам и брокерским фирмам для исполнения ордеров.
- **Сторонние аналитические инструменты**: Поддерживает интеграцию с различными сторонними аналитическими инструментами и инструментами управления рисками.
- **Пользовательские решения**: API и SDK для интеграции возможностей Deltix в пользовательские торговые решения и приложения.

## Сообщество и поддержка
Deltix предлагает обширную поддержку через подробную документацию, руководства пользователя, обучающие программы и специализированную команду поддержки. Они также предоставляют профессиональные услуги для помощи клиентам во внедрении и настройке своих решений.

## Случаи использования
- **Хедж-фонды**: Используется хедж-фондами для разработки и исполнения сложных стратегий количественной торговли.
- **Проприетарные торговые фирмы**: Поддерживает проприетарные торговые фирмы надежным бэктестингом, аналитикой в реальном времени и исполнением с низкой задержкой.
- **Управляющие активами**: Помогает управляющим активами оптимизировать свои торговые стратегии и эффективно управлять портфелями.
- **Исследовательские институты**: Используется исследовательскими институтами для количественных финансовых исследований и анализа.
- **Пользовательские торговые решения**: Позволяет разработчикам и фирмам создавать пользовательские торговые решения с помощью API и SDK Deltix.

Комплексный набор инструментов и услуг Deltix делает ее предпочтительным выбором для фирм, занимающихся количественными исследованиями, алгоритмической торговлей и стратегиями систематической торговли.
