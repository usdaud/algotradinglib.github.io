# Binance.US

Binance.US — это рынок цифровых активов, который предлагает безопасный, быстрый и надежный опыт торговли криптовалютами для пользователей в Соединенных Штатах. Запущенная в сентябре 2019 года, Binance.US была разработана для соответствия регуляциям США и предоставления доступной платформы для резидентов США для торговли различными криптовалютами. Платформа является партнером глобальной криптовалютной биржи Binance, которая является одной из крупнейших и наиболее авторитетных бирж в мире.

## Основание и история

Binance.US была создана для обслуживания американских пользователей, которые столкнулись с нормативными проблемами на глобальной платформе Binance. Биржа управляется BAM Trading Services и имеет лицензионное соглашение с Binance, получая преимущества от передовых технологий Binance, мер безопасности и ликвидности.

## Функции

### Широкий спектр криптовалют

Binance.US предлагает торговлю широким спектром цифровых активов. По последним данным пользователи могут торговать более чем 50 различными криптовалютами, включая:

- Bitcoin (BTC)
- Ethereum (ETH)
- Binance Coin (BNB)
- Tether (USDT)
- Cardano (ADA)
- Solana (SOL)
- Polkadot (DOT)

### Торговые пары

Платформа поддерживает множество торговых пар, которые включают USD, USDT (Tether), BTC и другие значимые криптовалюты, позволяя трейдерам эффективно объединять различные активы в пары.

### Безопасность

Безопасность имеет первостепенное значение на Binance.US, и биржа использует множество уровней защиты, включая:

- Продвинутое шифрование: Отраслевое стандартное AES-256 шифрование для защиты данных.
- Многофакторная аутентификация (MFA): Опции для аутентификации Google или SMS-2FA для безопасности счета.
- Холодное хранение: Большинство пользовательских фондов хранится в холодных кошельках, которые отключены от интернета для предотвращения взлома.

### Нормативное соответствие

Binance.US работает в соответствии с регуляциями США и зарегистрирована как Money Services Business (MSB) в FinCEN (Financial Crimes Enforcement Network). Платформа также использует продвинутые процедуры KYC (Знай своего клиента) и AML (Противодействие отмыванию денег) для предотвращения незаконных действий.

### Пользовательский интерфейс

Пользовательский интерфейс на Binance.US разработан так, чтобы быть легким в навигации для начинающих трейдеров, предлагая при этом продвинутые функции для профессиональных трейдеров.

- Базовый режим: Для новичков, простые опции покупки/продажи.
- Продвинутый режим: Для опытных трейдеров, включает графики, множество типов ордеров и углубленные инструменты анализа рынка.

## Комиссии

### Торговые комиссии

Одной из привлекательных особенностей Binance.US является её конкурентная структура комиссий. Торговые комиссии разработаны так, чтобы быть доступными по сравнению с другими биржами, базирующимися в США. Стандартная торговая комиссия начинается с 0,1%, и пользователи могут получить скидки на эти комиссии, держа Binance Coin (BNB) на своих счетах.

### Комиссии за депозиты и выводы

Депозиты в USD через ACH и банковские переводы обычно бесплатны, хотя некоторые финансовые учреждения могут взимать свои комиссии. Комиссии за вывод средств зависят от типа криптовалюты и комиссий сети.

## Поддерживаемые штаты

По последним данным Binance.US доступна в большинстве штатов США, но исключена из нескольких из-за специфических регуляций штата (например, Нью-Йорк).

## API для алгоритмической торговли

Binance.US предлагает доступ к API (Application Programming Interface), который популярен среди алгоритмических трейдеров и разработчиков. API позволяет программную торговлю и интеграцию с пользовательскими торговыми алгоритмами. Ключевые функции API Binance.US включают:

- RESTful конечные точки: Для общих торговых действий, таких как запрос данных, размещение сделок и управление счетом.
- WebSocket: Потоковая передача данных в реальном времени для рыночных данных и обновлений счета.
- Публичные конечные точки: Для получения данных, таких как сводка рынка и исторические данные.
- Приватные конечные точки: Для выполнения сделок, управления информацией о счете и мониторинга портфелей.

### Пример вызова API

Пример RESTful API вызова для размещения ордера может выглядеть следующим образом:

```markdown
POST /api/v3/order
Params:
    symbol (required) - String, e.g., 'BTCUSDT'
    side (required) - String, 'BUY' or 'SELL'
    type (required) - String, order type, e.g., 'LIMIT'
    timeInForce (optional) - String, e.g., 'GTC'
    quantity (required) - Decimal, e.g., '1.0'
    price (optional) - Decimal, e.g., '50000.0'
Headers:
    X-MBX-APIKEY - Your API key
```

Этот запрос сгенерирует ответ, содержащий детали размещенного ордера, включая ID ордера, статус и историю исполнения.

## Мобильное приложение

Binance.US также предлагает мобильное приложение, доступное как для iOS, так и для Android. Приложение включает:

- Торговля в реальном времени.
- Ценовые оповещения и push-уведомления.
- Безопасный вход с биометрией.
- Инструменты аналитики и построения графиков рынка.

## Образование и ресурсы

Понимая, что образование жизненно важно для роста пользователей, Binance.US предоставляет множество ресурсов, таких как:

- Блог: Регулярные обновления о последних тенденциях, рыночных новостях и образовательных статьях.
- Академия: Учебные пособия и объяснения, направленные на помощь как новичкам, так и продвинутым пользователям в улучшении их знаний.
- Поддержка клиентов: Надежная система поддержки, включая живой чат, поддержку по электронной почте и обширный раздел с часто задаваемыми вопросами.

## Стейкинг

Binance.US предлагает услуги стейкинга, позволяя пользователям получать вознаграждения за поддерживаемые криптовалюты, храня их в кошельках Binance.US. Эта функция поддерживает различные криптовалюты с доказательством доли (PoS) и делегированным доказательством доли (DPoS), такие как:

- Tezos (XTZ)
- Cosmos (ATOM)
- VeChain (VET)

Стейкинг приносит доходность на основе периода блокировки и стейканной суммы, обеспечивая дополнительный поток дохода для пользователей.

## Институциональные услуги

В дополнение к индивидуальным пользовательским счетам, Binance.US также обслуживает институциональных клиентов. Эти услуги включают:

- VIP-торговля: Пользовательские структуры комиссий и приоритетная поддержка для институциональных и высокообъемных трейдеров.
- OTC-торговля: Внебиржевая торговля для крупных транзакций для минимизации влияния на рынок.
- Институциональный API: Расширенные возможности API для поддержки высокочастотной торговли и сложных стратегий институциональных клиентов.

## Регуляторная среда и юридические вызовы

С момента своего создания Binance.US ориентировалась в сложной регуляторной среде, присущей рынку США. Она предприняла усилия для соответствия как федеральным, так и государственным регуляциям, в определенной степени успешно, хотя столкнулась со своей долей вызовов и критики.

### Юридические проблемы и критика

В октябре 2021 года появились сообщения о том, что Binance.US находится под расследованием CFTC (Commodity Futures Trading Commission) по поводу инсайдерской торговли и манипулирования рынком без необходимых раскрытий. Хотя эти обвинения не привели к окончательным юридическим действиям, они подчеркивают продолжающийся контроль, с которым сталкиваются криптовалютные биржи.

Кроме того, в 2022 году была инициирована серия коллективных исков против биржи, обвиняющих её в введении клиентов в заблуждение относительно безопасности и целостности рынка.

## Будущие перспективы и разработки

Binance.US продолжает развиваться и развиваться в ответ на рыночные требования и регуляторные изменения. Будущие планы включают введение более продвинутых торговых функций, расширение разнообразия поддерживаемых криптовалют и постоянное улучшение протоколов безопасности. Платформа также изучает возможность предложения фьючерсов и маржинальной торговли в ожидании регуляторного одобрения.

## Заключение

Binance.US быстро зарекомендовала себя как ведущая криптовалютная биржа, базирующаяся в США, используя технологии, безопасность и ликвидность своей родительской платформы, Binance. С конкурентными комиссиями, широким спектром криптовалют, надежными мерами безопасности и соблюдением регуляций США, Binance.US служит надежной платформой как для начинающих, так и для профессиональных трейдеров.
