# Barchart

Barchart является ведущим поставщиком финансовых данных, аналитики и технологических решений для финансовых рынков и бизнеса. Известная своим обширным охватом рыночных данных и удобными платформами, Barchart обслуживает широкий круг клиентов от индивидуальных инвесторов до крупных финансовых институтов.

## Основные продукты и услуги

### 1. Услуги рыночных данных

- Данные в режиме реального времени и с задержкой для акций, фьючерсов, форекса и криптовалют
- Данные на конец дня (EOD)
- Архивы исторических данных
- Фундаментальные данные для акций и ETF
- Технические индикаторы и аналитика

### 2. Barchart.com

- Публичный финансовый портал
- Бесплатные и премиальные инструменты рыночных данных и анализа
- Новости и рыночные комментарии
- Настраиваемые списки наблюдения и портфели

### 3. Barchart Trader

- Продвинутая платформа для построения графиков
- Котировки в режиме реального времени с потоковой передачей
- Инструменты анализа опционов
- Технические исследования и инструменты рисования
- Мобильные приложения для iOS и Android

### 4. API решения

- API рыночных данных
- Торговые API
- Аналитические и вычислительные API
- WebSocket API для потоковой передачи данных в режиме реального времени

### 5. cmdtyView

- Исследовательская и аналитическая платформа, ориентированная на сырьевые товары
- Цены в режиме реального времени, новости и фундаментальные показатели для сельскохозяйственных и энергетических рынков
- Собственные индексы и аналитика

### 6. Корпоративные решения

- Потоки данных для внутренних приложений
- Решения по созданию веб-сайтов и мобильных приложений под собственным брендом
- Услуги пользовательской разработки
- Управляемые услуги для управления данными

Barchart зарекомендовал себя как универсальный и надежный поставщик финансовых данных, обслуживая широкий спектр пользователей от индивидуальных розничных инвесторов до крупных финансовых институтов. Его сочетание всеобъемлющего охвата данных, удобных инструментов и гибких корпоративных решений сделало его популярным выбором в сфере финансовых данных и аналитики. Постоянные инновации компании в таких областях, как данные о сырьевых товарах, услуги API и аналитика на основе машинного обучения, продолжают укреплять ее позиции на рынке.
