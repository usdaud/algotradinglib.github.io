# AmiBroker

## Обзор
AmiBroker - мощное программное обеспечение для технического анализа и трейдинга, позволяющее трейдерам разрабатывать, тестировать и внедрять торговые стратегии. Он известен гибкостью, скоростью и широким набором функций, что делает его популярным выбором среди розничных трейдеров и профессиональных инвесторов.

## Возможности
1. **Графики и анализ**: расширенные возможности построения графиков с большим набором технических индикаторов и инструментов рисования.
2. **Язык формул (AFL)**: AmiBroker Formula Language (AFL) для создания пользовательских индикаторов, бэктестинга стратегий и автоматизации торговых систем.
3. **Бэктестинг и оптимизация**: высокопроизводительный движок бэктестинга, позволяющий тщательно тестировать и оптимизировать торговые стратегии на исторических данных.
4. **Данные в реальном времени**: поддержка потоковых данных и подключение к множеству поставщиков для анализа рынка в реальном времени.
5. **Управление портфелем**: инструменты анализа и управления на уровне портфеля, включая размер позиции и управление рисками.
6. **Автоматизация**: функции автоматического исполнения торговых стратегий без ручного вмешательства.
7. **Расширяемость**: поддержка сторонних плагинов и пользовательских скриптов для расширения функциональности.

## Ключевые компоненты
- **Движок графиков**: интерактивные графики с настраиваемыми индикаторами и исследованиями.
- **AFL (AmiBroker Formula Language)**: фирменный язык сценариев для разработки стратегий, пользовательских индикаторов и бэктестинга.
- **Инструменты анализа**: комплексные средства технического анализа, распознавания паттернов и генерации сигналов.
- **Бэктестинг и walk-forward тестирование**: надежный движок бэктестинга с поддержкой walk-forward тестирования для оценки устойчивости стратегии.
- **Оптимизация**: оптимизатор на основе генетических алгоритмов для поиска лучших параметров торговых стратегий.
- **Интеграция с брокерами**: интерфейсы для работы с различными брокерами для живого исполнения ордеров.

## Интеграции
AmiBroker интегрируется с множеством источников данных и брокерских платформ, обеспечивая цельный торговый процесс. Среди заметных интеграций:

- **Провайдеры рыночных данных**: поддержка данных в реальном времени и истории от поставщиков вроде Interactive Brokers, eSignal и IQFeed.
- **Брокеры**: совместимость с брокерами, такими как Interactive Brokers, для прямого исполнения сделок.
- **Сторонние плагины**: широкая поддержка сторонних плагинов и API для расширения возможностей и интеграции с другими торговыми инструментами.
