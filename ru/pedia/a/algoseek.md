# AlgoSeek

AlgoSeek - ведущий поставщик высококачественных финансовых данных и аналитических решений, специализирующийся на альтернативных данных для количественных исследований и торговых стратегий. Компания фокусируется на предоставлении детализированных, чистых и комплексных наборов данных для финансовых организаций, хедж-фондов и количественных трейдеров.

## Основные предложения

### 1. Исторические рыночные данные

- Тиковые данные по акциям, опционам и фьючерсам
- Полная глубина стакана
- Восстановленные снимки книги заявок
- Данные сделок и котировок с микросекундной точностью

### 2. Альтернативные данные

- Сигналы розничной торговли
- Потоки опционов
- Индикаторы короткого интереса
- Анализ тональности соцсетей
- Наборы данных веб-скрейпинга

### 3. Данные корпоративных действий

- Полная история корпоративных действий
- Объявления и выплаты дивидендов
- Сплиты и слияния
- Эмиссии прав и выделения (spinoffs)

### 4. Справочные данные

- Файлы security master
- Информация о корпоративной иерархии
- Данные классификации отраслей
- Фундаментальные финансовые данные

### 5. Аналитика и производные данные

- Метрики ликвидности
- Индикаторы волатильности
- Классификация сделок (инициатор покупки/продажи)
- Обнаружение аномальных объемов

## Ключевые особенности

### Качество и чистота данных

- Строгие процессы очистки и нормализации
- Обработка корпоративных действий и смен тикеров
- Кросс-проверка по нескольким источникам
- Единые форматы данных по классам активов

### Кастомизация и доставка

- Гибкие способы доставки данных (API, FTP, облачное хранилище)
- Пользовательские выборки и преобразования
- Поддержка разных форматов файлов (CSV, Parquet, HDF5 и др.)
- Потоки данных в реальном времени и история

### Поддержка исследований и бэктестинга

- Примерные наборы данных для оценки
- Среды бэктестинга, совместимые с популярными платформами
- Документация и примеры кода для интеграции

### Соответствие и безопасность

- Данные собираются и распространяются с соблюдением договоров бирж
- Безопасная передача и хранение данных
- Анонимизация чувствительной информации при необходимости

## Технологическая инфраструктура
