# TradeStation

TradeStation — это комплексная электронная торговая платформа, разработанная TradeStation Group, Inc. Известная своими мощными аналитическими инструментами и продвинутыми функциями, TradeStation обслуживает активных трейдеров, искушённых инвесторов и институциональных участников по различным классам активов.

## Основные функции

### 1. Продвинутые графики и анализ

- Множество типов графиков и таймфреймов
- Более 270 встроенных технических индикаторов
- Создание пользовательских индикаторов с использованием EasyLanguage
- Инструменты рисования и распознавание паттернов
- Потоковые данные в реальном времени
- Поддержка нескольких мониторов
- Radar Screen для сканирования рынка в реальном времени

### 2. Программирование EasyLanguage

- Проприетарный язык скриптов для разработки пользовательского анализа и стратегий
- Возможность создавать пользовательские индикаторы, стратегии и функции
- Обширная библиотека готовых скриптов и стратегий
- Обмен через сообщество и торговая площадка для скриптов EasyLanguage

### 3. Тестирование и оптимизация стратегий

- Продвинутый движок бэктестинга
- Walk-forward оптимизация
- Анализ Монте-Карло
- Тестирование портфеля из нескольких стратегий
- Исторический повтор рынка для проверки стратегии

### 4. Исполнение ордеров и управление торговлей

- Поддержка акций, ETF, опционов, фьючерсов, форекса и криптовалют
- Продвинутые типы ордеров (включая условные ордера)
- Матричный (лестничный) торговый интерфейс
- Отслеживание позиций и управление рисками в реальном времени
- Настраиваемые оповещения и уведомления

### 5. OptionStation Pro

- Специализированная платформа для анализа опционов
- Цепочки опционов с греками в реальном времени
- Инструменты анализа вероятности
- Графики риска и моделирование позиций
- Анализ перекоса волатильности

### 6. Автоматизированная торговля

- Полностью автоматизированное исполнение стратегии
- API-доступ для пользовательских торговых приложений
- Поддержка высокочастотных торговых стратегий

### 7. Данные и исследования

- Рыночные данные в реальном времени
- Интеграция фундаментальных данных и новостей
- Экономический календарь
- Отчёты о доходах и корпоративные действия

### 8. Мобильная торговля

- Приложения для iOS и Android с большинством функций десктопа
- Синхронизированные списки наблюдения и оповещения на всех устройствах
- Полные торговые возможности на мобильных устройствах

## Версии платформы

1. **TradeStation Desktop**: Полнофункциональное настольное приложение
2. **Web Trading**: Браузерная платформа с основными функциями
3. **Мобильные приложения**: Приложения для iOS и Android для торговли в пути
4. **OptionStation Pro**: Специализированный интерфейс для торговли опционами

## Уникальные инструменты

### 1. RadarScreen

- Сканер рынка в реальном времени
- Настраиваемые столбцы и оповещения
- Возможность запуска анализа EasyLanguage на нескольких символах одновременно

### 2. Matrix

- Продвинутый интерфейс ввода ордеров
- Визуализация глубины рынка
- Возможности торговли в один клик

### 3. Hot Lists

- Готовые и настраиваемые скринеры акций
- Обновление в реальном времени на основе рыночных условий

### 4. Time and Sales с данными Level II

- Подробная информация о сделках и котировках
- Анализ глубины рынка

## Данные и исследования

- Могут применяться сборы за данные для определённых пакетов рыночных данных
- Доступны скидки по объёму для активных трейдеров

TradeStation известна своими сложными инструментами, которые обслуживают серьёзных трейдеров и инвесторов. Её сочетание продвинутых аналитических возможностей, надёжных инструментов разработки и тестирования стратегий и поддержки нескольких классов активов делает её популярным выбором для тех, кому требуется профессиональная торговая платформа. Постоянная эволюция и адаптация платформы к новым рыночным тенденциям, таким как торговля криптовалютами, обеспечивают её актуальность в постоянно меняющемся ландшафте электронной торговли.
