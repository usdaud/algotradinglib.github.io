# TradingView

TradingView - это облачная платформа для построения графиков и социальных сетей для трейдеров и инвесторов. Известная своим удобным интерфейсом, продвинутыми возможностями построения графиков и яркими сообществом, TradingView стала одним из самых популярных инструментов финансового анализа во всем мире.

## Ключевые функции

### 1. Продвинутое построение графиков

- HTML5 графики с более чем 50 инструментами рисования
- Более 100 встроенных технических индикаторов
- Создание пользовательских индикаторов с использованием Pine Script
- Анализ на нескольких таймфреймах
- Графики сравнения для нескольких активов
- Видимость торговли в расширенные часы
- Инструменты профиля объема и рыночного профиля

### 2. Скринеры и рыночный анализ

- Скринеры акций, Forex и криптовалют
- Экономический календарь
- Настраиваемые списки наблюдения
- Тепловые карты для визуального анализа рынка
- Сводка технического анализа и рейтинги

### 3. Социальные сети

- Платформа для обмена идеями и публикаций
- Подписка на других трейдеров и аналитиков
- Комментарии и обсуждения графиков
- Система репутации (на основе точности идей)
- Функциональность приватного чата

### 4. Pine Script

- Собственный скриптовый язык для пользовательских индикаторов и стратегий
- Обширная библиотека скриптов, созданных сообществом
- Возможности бэктестинга торговых стратегий

### 5. Покрытие данных

- Данные в реальном времени и исторические данные для:
 - Акций и ETF
 - Валютных пар Forex
 - Криптовалют
 - Фьючерсов и товаров
 - Облигаций и экономических данных
- Глобальное рыночное покрытие (более 50 бирж)

### 6. Оповещения и уведомления

- Ценовые оповещения
- Оповещения технических индикаторов
- Оповещения пользовательских условий с использованием Pine Script
- Push-уведомления на мобильные устройства
- Оповещения по электронной почте и SMS (на более высоких тарифах)

### 7. Мобильные и планшетные приложения

- Приложения для iOS и Android
- Синхронизированные списки наблюдения и графики на всех устройствах
- Данные в реальном времени и оповещения на мобильных устройствах

### 8. Расширения для браузеров

- Расширения для Chrome и Firefox для быстрого доступа к графикам

### 9. Поддержка API и виджетов

- Библиотека графиков для интеграции с онлайн-платформами
- REST API для доступа к данным
- Виджеты для отображения графиков на внешних веб-сайтах

## Версии платформы

1. **Веб-платформа**: Основное приложение на основе браузера
2. **Десктопное приложение**: Загружаемая версия для Windows и macOS
3. **Мобильные приложения**: Версии для iOS и Android

## Уровни подписки

1. **Basic (Бесплатный)**
 - Ограниченные функции и отложенные данные
 - Доступ к сообществу и базовому построению графиков

2. **Pro**
 - Данные в реальном времени для выбранных рынков
 - Несколько графиков на макет
 - Без рекламы
 - Данные торговли в расширенные часы

3. **Pro+**
 - Дополнительные данные и функции
 - Экзотические внутридневные графики
 - Интервалы на основе секунд
 - Больше серверных оповещений

4. **Premium**
 - Все функции разблокированы
 - Приоритетная поддержка
 - Неограниченные серверные оповещения
