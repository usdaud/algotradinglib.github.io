# Бенчмарковые курсы WM/Reuters

Бенчмарковые курсы WM/Reuters - это важные референсные данные обменных курсов для финансовой индустрии.

## Обзор

Предоставляют надежную, прозрачную и независимую ссылку на глобальные валютные курсы.

## Применения

- Оценка портфеля
- Измерение производительности
- Исполнение торговли
- Соблюдение регулирования

## Ключевые времена фиксинга

- 16:00 по лондонскому времени
- 11:00 по нью-йоркскому времени
- 19:00 по токийскому времени

## Заключение

WM/Reuters формируют важную основу для глобального валютного рынка.
