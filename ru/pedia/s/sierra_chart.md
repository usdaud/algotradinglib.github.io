# Sierra Chart

Sierra Chart — это высокотехнологичная профессиональная программная платформа для торговли и построения графиков, предназначенная для серьезных трейдеров, финансовых аналитиков и институциональных клиентов. Известная своей надежностью, гибкостью и обширным функционалом, Sierra Chart зарекомендовала себя как комплексное решение для тех, кто требует расширенных торговых возможностей по множеству классов активов.

## Основной функционал

### 1. Графики и технический анализ

Возможности построения графиков Sierra Chart являются одними из самых продвинутых в отрасли:

- **Типы графиков**: Предлагает широкий спектр типов графиков, включая японские свечи, бары, линии, крестики-нолики, Ренко, Каги и другие.
- **Временные интервалы**: Поддерживает пользовательские временные интервалы от тиковых до годовых графиков, с возможностью создания нестандартных периодов.
- **Наложение нескольких символов**: Позволяет сравнивать несколько символов на одном графике, что необходимо для корреляционного анализа.
- **Инструменты рисования**: Обширный набор инструментов рисования, включая линии тренда, уровни Фибоначчи, паттерны волн Эллиотта и пользовательские фигуры.
- **Индикаторы**: Включает более 300 встроенных технических индикаторов, охватывающих все от простых скользящих средних до сложных алгоритмических исследований.
- **Пользовательские индикаторы**: Позволяет создавать пользовательские индикаторы с использованием C++ или ACSIL (Advanced Custom Study Interface Language), предоставляя неограниченные аналитические возможности.
- **Обновления в реальном времени**: Все графики и исследования обновляются в реальном времени, гарантируя трейдерам доступ к актуальной рыночной информации.

### 2. Исполнение ордеров и торговля

Sierra Chart предоставляет комплексный торговый интерфейс:

- **Типы ордеров**: Поддерживает широкий спектр типов ордеров, включая рыночные, лимитные, стоп-ордера, OCO (один отменяет другой), скользящие стопы и другие.
- **Прямой доступ к рынку (DMA)**: Предлагает прямые подключения к многочисленным биржам, обеспечивая более быстрое исполнение и сокращение проскальзывания.
- **DOM Trading**: Имеет динамическое окно глубины рынка (DOM) для быстрого размещения ордеров и анализа глубины рынка.
- **Управление позициями**: Отслеживание позиций, P&L и метрик риска в режиме реального времени по нескольким счетам и инструментам.
- **Поддержка нескольких брокеров**: Интегрируется с различными FCM и брокерами, позволяя трейдерам использовать предпочитаемые сервисы исполнения.

### 3. Автоматизированная торговля

Платформа предлагает надежные возможности для автоматизированной и алгоритмической торговли:

- **Разработка стратегий**: Трейдеры могут создавать сложные торговые стратегии с использованием C++ или ACSIL.
- **Движок бэктестинга**: Комплексные возможности бэктестинга позволяют оптимизировать стратегии и анализировать историческую производительность.
- **Исполнение стратегий в реальном времени**: Автоматизированные стратегии могут исполняться в реальных рыночных условиях.
- **Событийно-ориентированная архитектура**: Система реагирует на рыночные события в реальном времени, позволяя реализовывать высокочастотную торговлю.

### 4. Управление данными и анализ

Sierra Chart превосходно справляется с обработкой и анализом рыночных данных:

- **Множество источников данных**: Интегрируется с многочисленными поставщиками данных для получения данных в реальном времени и исторических данных по различным классам активов.
- **Запись данных**: Возможность записывать и хранить тиковые данные для последующего анализа или воспроизведения рынка.
- **Воспроизведение рынка**: Сложный функционал воспроизведения рынка позволяет трейдерам детально изучать исторические рыночные условия.
- **Анализ объема**: Продвинутые инструменты профиля объема и профиля рынка для углубленного анализа структуры рынка.
- **Time and Sales**: Детальное окно время и продажи для анализа на уровне транзакций.

### 5. Управление рисками

Платформа включает несколько функций для эффективного управления рисками:

- **Размер позиции**: Инструменты для расчета и внедрения соответствующих размеров позиций на основе баланса счета и толерантности к риску.
- **Управление стоп-лоссами**: Автоматическое размещение стоп-лоссов и возможности скользящих стопов.
- **Расчет маржи**: Расчет требований маржи в реальном времени для предотвращения чрезмерного кредитного плеча.
- **Сценарный анализ**: Инструменты для анализа потенциальных результатов сделок при различных рыночных условиях.

## Технические характеристики

- **Производительность**: Известна своей высокоскоростной производительностью, способна обрабатывать большие объемы данных и сложные вычисления с минимальной задержкой.
- **Стабильность**: Надежная архитектура обеспечивает стабильность даже в стрессовых рыночных условиях.
- **Настройка**: Практически каждый аспект платформы может быть настроен, от макетов графиков до сочетаний клавиш.
- **Интеграция API**: Предлагает API для интеграции с внешними системами и источниками данных.
- **Поддержка нескольких мониторов**: Разработана для бесшовной работы на нескольких мониторах для расширенной торговой установки.

## Совместимость платформы и требования

- **Операционные системы**:
 - Windows 7, 8, 10, 11 (32-битные и 64-битные версии)
 - Linux (через Wine)
- **Аппаратные требования**:
 - Минимум: Двухъядерный процессор, 4ГБ RAM, 1ГБ свободного дискового пространства
 - Рекомендуется: Четырехъядерный процессор или лучше, 8ГБ+ RAM, SSD для улучшенной производительности

## Лицензирование и ценообразование

Sierra Chart предлагает гибкие варианты лицензирования для различных потребностей трейдеров:

- **Планы подписки**: Доступны месячные и годовые планы со скидками при длительных обязательствах.
- **Постоянная лицензия**: Вариант единовременной покупки для трейдеров, предпочитающих владение вместо подписки.
- **Дополнения**: Дополнительные сборы могут применяться для определенных источников данных, подключений к биржам или расширенных функций.
- **Бесплатная пробная версия**: Предлагает пробный период для пользователей, чтобы протестировать платформу перед принятием решения.
