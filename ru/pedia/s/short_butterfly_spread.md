# Короткий спред «бабочка»

## Введение

Короткий спред «бабочка» — это опционная торговая стратегия, которая используется когда трейдер ожидает минимального движения цены базового актива. Эта стратегия включает одновременную покупку и продажу опционов с тремя различными ценами исполнения.

## Компоненты короткого спреда «бабочка»

### Ключевые элементы:

1. **Базовый актив**: Финансовый инструмент, на который выписаны опционы.
2. **Опционы**: Финансовые производные.
3. **Цены исполнения**: Цены, по которым опционы могут быть исполнены.
4. **Дата экспирации**: Дата, на которую истекает опцион.

### Построение:

- **Продать один колл с нижним страйком**
- **Купить два колла со средним страйком**
- **Продать один колл с верхним страйком**

## Характеристики выплаты

### Максимальный убыток:
Возникает, если цена ниже нижней цены исполнения или выше верхней на момент экспирации.

### Максимальная прибыль:
Достигается, когда цена находится на средней цене исполнения.

### Точки безубыточности:
- Нижняя = Нижняя цена + Чистая премия
- Верхняя = Верхняя цена - Чистая премия

## Преимущества
- Ограниченный риск
- Подходит для низкой волатильности
- Определенная прибыль и убыток

## Недостатки
- Сложная стратегия
- Ограниченный потенциал прибыли

## Заключение

Короткий спред «бабочка» — это тонкая стратегия для трейдеров, работающих с опционами в среде с низкой волатильностью. При правильном исполнении может быть прибыльной для трейдеров, ожидающих небольшого движения цены.
