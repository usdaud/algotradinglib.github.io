# Sterling Trader

Sterling Trader — это набор профессиональных электронных торговых платформ, разработанных для активных трейдеров, проприетарных торговых групп и институциональных клиентов. Разработанные Sterling Trading Tech, эти платформы известны своими надежными функциями, надежностью и опциями настройки.

## Ключевые продукты

1. Sterling Trader® Pro
2. Sterling Trader® Elite
3. Sterling Trader® Web
4. Sterling Trader® Mobile

## Sterling Trader® Pro

Флагманский продукт линейки, Sterling Trader Pro — это мощная брокерско-нейтральная платформа, разработанная для профессиональных трейдеров акциями.

### Ключевые особенности:

1. **Расширенный ввод ордеров**
 - Множественные типы ордеров (рыночные, лимитные, стоп и т.д.)
 - Торговля в один клик
 - Настраиваемые горячие клавиши
 - Возможности корзинной торговли

2. **Рыночные данные в реальном времени**
 - Котировки уровня I и уровня II
 - Окно Time & Sales
 - Настраиваемые списки наблюдения
 - Интеграция новостей в реальном времени

3. **Графики**
 - Множественные типы графиков и таймфреймы
 - Технические индикаторы и инструменты рисования
 - Настраиваемые макеты

4. **Управление рисками**
 - Отслеживание позиций в реальном времени
 - Мониторинг прибылей и убытков
 - Настраиваемые оповещения и уведомления

5. **Маршрутизация ордеров**
 - Прямой доступ к рынку (DMA)
 - Интеллектуальная маршрутизация ордеров
 - Доступ к темным пулам и ECN

6. **Настройка**
 - Полностью настраиваемый интерфейс
 - API для пользовательских дополнений и интеграций

7. **Инструменты соответствия**
 - Функции надзора за торговлей
 - Возможности отчетности для регулятивных требований

## Sterling Trader® Elite

Расширенная версия Pro, разработанная для высокочастотных трейдеров и институциональных клиентов, требующих сверхнизкой латентности.

### Дополнительные функции:
- Опции колокации для снижения латентности
- Расширенные возможности алгоритмической торговли
- Более широкий доступ к API

## Sterling Trader® Web

Браузерная версия, предлагающая основные торговые функциональности без необходимости установки программного обеспечения.

### Ключевые особенности:
- Доступ с любого устройства с подключением к Интернету
- Котировки и графики в реальном времени
- Базовый ввод ордеров и управление позициями

## Sterling Trader® Mobile

Мобильное приложение для iOS и Android, позволяющее трейдерам отслеживать рынки и управлять позициями в движении.

### Ключевые особенности:
- Рыночные данные и списки наблюдения в реальном времени
- Базовый ввод ордеров и управление позициями
- Push-уведомления для оповещений

## Практический чек-лист
- Определите временной горизонт для Sterling Trader и рыночный контекст.
- Определите входные данные, которым вы доверяете, такие как цена, объем или даты расписания.
- Напишите четкое правило входа и выхода перед вложением капитала.
- Определите размер позиции так, чтобы одна ошибка не повредила счету.
- Документируйте результат для улучшения повторяемости.

## Распространенные ошибки
- Рассматривать Sterling Trader как самостоятельный сигнал вместо контекста.
- Игнорирование ликвидности, спредов и трения исполнения.
- Использование правила на другом таймфрейме, чем тот, для которого оно было разработано.
- Переобучение на небольшой выборке прошлых примеров.
- Предположение о том же поведении при аномальной волатильности.

## Данные и измерения
Хороший анализ начинается с согласованных данных. Для Sterling Trader подтвердите источник данных, часовой пояс и частоту выборки. Если концепция зависит от расчетных или календарных дат, согласуйте календарь с правилами биржи. Если она зависит от ценового действия, рассмотрите использование скорректированных данных для учета корпоративных действий.
